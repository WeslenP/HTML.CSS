Usanso Listas OL e UL

Lista OL

OL são Listas ordenadas que precisam de uma ordem, podemos adicinar o type com 1, A, a, i e I e també adicionar o start para indicar onde ela deve começar

  1. acorda
  2. tomar cafe
  3. falar com jose
  4. escovar os dentes
  5. ir para escola
  6. almoçar
  7. ir para academia
  8. levar o cão para passear
  9. jantar

Lista UL

a lista não ordenada, são aquelas onde não se precisa enumerar podemos adicionar o type com disc ,circle e square

  • pão
  • leite
  • feijão
  • café
  • arroz

Listas mistas

podemos criar listas mistas com OL e UL

  • Casa onde morei
    1. Vargem grande
  • Paises onde quero morar
    1. China
    2. Tailandia
    3. Coreia
